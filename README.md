# SIMPLE RAILS 5 REST API

Example of a basic REST API to perform CRUD functions. Uses Sqli3 by default. Please add your own database and credentials

## INSTALL DEPENDENCIES
```bash
$ bundle install
```

## RUN MIGRATION
```bash
$ rails db:migrate
```
## RUN SEEDS
```bash
$ rails db:seed
```
## RUN SERVER
```bash
$ rails s
```
